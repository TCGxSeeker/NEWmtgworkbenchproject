# Import Export Requirements

## Import

Core import workflows:

- Paste plain text decklist.
- Load local plain text file.
- Load simple CSV decklist.
- Preserve commander and maybeboard sections when marked.
- Report unknown cards instead of guessing.

## Export

Core export workflows:

- Plain text with quantities and card names.
- Machine-readable JSON for internal state.
- Optional CSV export for deck rows.

## Save And Load

- Save local deck workspace state.
- Load saved deck workspace state.
- Preserve view/group/sort/filter preferences separately from deck card facts.
- Record local card snapshot metadata used for normalization.

## Non-Goals

- No live deck-hosting integration in the first product surface.
- No account system requirement.
- No purchase automation.
