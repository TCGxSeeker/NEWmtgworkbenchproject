# Role Taxonomy Seed

## Purpose

Define the roles cards can fill in a specific deck context. Roles should describe what a card is doing in the deck, not just its printed card type.

## Fill In Later

- Ramp roles.
- Draw and card-flow roles.
- Interaction roles.
- Protection roles.
- Engine roles.
- Payoff roles.
- Win-condition roles.
- Flex roles.
- Role confidence rules.
