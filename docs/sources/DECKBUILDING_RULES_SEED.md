# Deckbuilding Rules Seed

## Purpose

Capture human-readable deckbuilding rules before converting them into testable local config files.

## Fill In Later

- Commander bracket assumptions.
- Power-level scoring rubric.
- Land count rules.
- Ramp, draw, interaction, protection, engine, payoff, and win-condition expectations.
- Deck construction warnings.
- Exceptions and approval requirements.

## Rule Draft Template

```text
Rule:
Applies when:
Reason:
Mechanical check:
Human validation required:
Example:
```
